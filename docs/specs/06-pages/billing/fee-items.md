## 状态机/流转

```
┌──────────┐
│  active  │ (启用，可在收费时选择)
└────┬─────┘
     │ 停用
     ▼
┌──────────┐
│ disabled │ (停用，不可在收费时选择)
└────┬─────┘
     │ 启用
     │
     └──────→ active
```

### 状态流转规则

| 当前状态 | 可流转至 | 触发条件 | 说明 |
|----------|----------|----------|------|
| `active` | `disabled` | 点击 [停用] | 停用后收费开单不显示此项目; 已有账单不受影响 |
| `disabled` | `active` | 点击 [启用] | 恢复可选状态 |

### 删除规则

| 条件 | 是否可删除 | 说明 |
|------|------------|------|
| 未被任何账单引用 | 可删除 | 物理删除 |
| 已被账单引用 | 不可删除 | 仅可停用; 删除按钮置灰并 Tooltip 提示 "该项目已被 {N} 条账单引用，无法删除" |

## 交互说明

### INT-01: 页面加载

| 步骤 | 行为 |
|------|------|
| 1 | 页面加载时请求分类树和费用项目列表 (默认按创建时间降序) |
| 2 | 并行请求: 分类树 + 项目列表 + 币种配置 |
| 3 | 左侧分类树默认选中 "全部" |
| 4 | 表格使用骨架屏 (Skeleton) 作为加载占位 |
| 5 | 检查 URL query 参数，如有 `category` 则自动选中对应分类 |

### INT-02: 分类树交互

| 步骤 | 行为 |
|------|------|
| 1 | 点击分类节点 → 自动刷新右侧表格; 高亮选中分类 |
| 2 | 点击 "全部" → 显示所有分类项目 |
| 3 | 分类节点右侧显示该分类项目计数 Badge |
| 4 | 右键分类节点 → 显示上下文菜单: 编辑分类 / 删除分类 / 新增子分类 |
| 5 | 点击 [+ 新分类] → 弹窗创建新分类; 支持父子层级 |

### INT-03: 搜索交互

| 步骤 | 行为 |
|------|------|
| 1 | 输入关键词，300ms 防抖后自动搜索 |
| 2 | 搜索时表格显示 loading 状态 |
| 3 | 搜索结果高亮匹配文字 |
| 4 | 搜索时自动切换分类树到 "全部"，搜索范围为全局 |
| 5 | 清空搜索框 → 恢复原分类筛选结果 |

### INT-04: 排序交互

| 步骤 | 行为 |
|------|------|
| 1 | 点击可排序列头切换排序: 无 → 升序 → 降序 → 无 |
| 2 | 排序图标: ⬆升序 / ⬇降序 / ⬍默认 |
| 3 | 仅支持单列排序 |
| 4 | 切换排序时保持当前页码 |

### INT-05: 新建/编辑项目

| 步骤 | 行为 |
|------|------|
| 1 | 点击 [新建项目] 打开右侧抽屉 (480px) |
| 2 | 新建时自动生成编码 (如 FI + 自增序号); 分类默认选中当前分类树节点 |
| 3 | 填写项目信息; 如选择 "是否套餐" → 展开套餐明细区域 |
| 4 | 套餐明细支持搜索选择子项目; 动态增删行 |
| 5 | 点击 [保存] → 校验表单; 成功 → Toast "项目已保存"; 列表刷新; 分类树计数更新 |
| 6 | 点击 [取消] → 如有修改弹出确认: "有未保存的修改，确定离开？" |
| 7 | 编辑时: 打开抽屉预填现有数据; 编码字段不可修改 |

### INT-06: 状态切换

| 步骤 | 行为 |
|------|------|
| 1 | 点击表格中状态 Badge → 直接切换状态 (启用↔停用) |
| 2 | 停用时弹出确认弹窗; 启用时直接切换 |
| 3 | 成功 → 行内状态 Badge 动画更新; Toast "{项目名}已{启用/停用}" |
| 4 | 失败 → Toast 错误提示; 状态不变 |

### INT-07: 导入功能

| 步骤 | 行为 |
|------|------|
| 1 | 点击 [导入] → 打开导入弹窗 |
| 2 | 提供模板下载链接: "下载导入模板 (.xlsx)" |
| 3 | 上传 Excel 文件 → 系统解析并显示预览 (前 10 行) |
| 4 | 校验: 编码唯一性、必填项、数据格式 |
| 5 | 显示校验结果: {N} 条可导入 / {M} 条有错误 |
| 6 | 点击 [确认导入] → 导入成功的记录; 错误记录可下载错误报告 |
| 7 | Toast "成功导入 {N} 条记录" |

### INT-08: 导出功能

| 步骤 | 行为 |
|------|------|
| 1 | 点击 [导出] 按钮 |
| 2 | 如有选中行 → 导出选中行; 无选中行 → 导出当前筛选结果全部数据 |
| 3 | 生成 Excel 文件 (.xlsx)，包含所有表格列 |
| 4 | 文件名格式: `费用项目_{YYYYMMDD_HHmmss}.xlsx` |
| 5 | 大数据量 (>1000条) 时显示导出进度条 |
| 6 | 导出完成 → Toast 提示 "导出成功" |

| 无费用项目 | 文件插画 + "暂无费用项目" + [新建项目] 按钮 |
| 搜索无结果 | 搜索插画 + "未找到匹配的费用项目" + "请尝试其他搜索词或调整筛选条件" |
| 筛选无结果 | 筛选插画 + "没有符合条件的费用项目" + [清除筛选] 链接按钮 |
| 分类下无项目 | 空分类插画 + "该分类下暂无费用项目" + [新建项目] 链接 |
| 套餐无子项目 | 套餐区域: "暂无套餐明细" + [添加子项目] 链接 |

## API 数据源

| 接口 | 方法 | 路径 | 请求参数 | 说明 |
|------|------|------|----------|------|
| 获取分类树 | GET | `/api/v1/fee-items/categories` | - | 分类树结构数据; 含每个分类的项目计数 |
| 创建分类 | POST | `/api/v1/fee-items/categories` | `name`, `parent_id` | 创建费用项目分类 |
| 更新分类 | PUT | `/api/v1/fee-items/categories/:id` | `name` | 更新分类名称 |
| 删除分类 | DELETE | `/api/v1/fee-items/categories/:id` | - | 仅空分类可删除 |
| 获取费用项目列表 | GET | `/api/v1/fee-items` | `keyword`, `category`, `status`, `is_package`, `price_min`, `price_max`, `sort_field`, `sort_order`, `page`, `page_size` | 分页获取费用项目列表 |
| 获取费用项目详情 | GET | `/api/v1/fee-items/:id` | - | 获取单个项目完整信息 (含套餐明细) |
| 创建费用项目 | POST | `/api/v1/fee-items` | `code`, `name`, `category`, `description`, `default_price`, `currency`, `unit`, `tax_rate`, `is_package`, `status`, `package_items[]` | 创建费用项目 |
| 更新费用项目 | PUT | `/api/v1/fee-items/:id` | 同创建参数 (code 不可改) | 更新费用项目 |
| 删除费用项目 | DELETE | `/api/v1/fee-items/:id` | - | 仅未被引用的项目可删除 |
| 批量更新状态 | PATCH | `/api/v1/fee-items/batch-status` | `ids[]`, `status` | 批量启用/停用 |
| 批量删除 | DELETE | `/api/v1/fee-items/batch` | `ids[]` | 批量删除; 跳过已被引用的 |
| 导入费用项目 | POST | `/api/v1/fee-items/import` | `file` (multipart) | Excel 批量导入 |
| 导出费用项目 | POST | `/api/v1/fee-items/export` | 同列表筛选参数 + `ids[]` (可选) | 导出 Excel 文件 |
| 下载导入模板 | GET | `/api/v1/fee-items/import-template` | - | 下载 Excel 导入模板 |
| 获取币种配置 | GET | `/api/v1/settings/currencies` | - | 支持的币种列表和汇率 |
