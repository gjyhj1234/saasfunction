# 患者详情

## 页面信息
- route: /patients/:id
- module: patient
- permission: patient:view
- layout: 顶部信息卡片 + 标签页

## 顶部患者信息卡片

| 元素 | 说明 |
|------|------|
| 患者头像/首字母 | 圆形头像 64x64 |
| 姓名 | 大字体加粗 |
| 患者编号 | #P-xxxxx |
| 会员等级 | 彩色标签 |
| 性别/年龄 | 男 28岁 |
| 手机号 | 带区号 |
| 标签组 | 所有标签横排 |
| 最近就诊 | 日期 |
| 欠费金额 | 红色高亮(有欠费时) |
| 累计消费 | 金额 |
| 编辑按钮 | 编辑基本信息 |
| 更多操作 | 下拉菜单: 打标签/关联家庭/设为黑名单 |

## 标签页

### Tab 1: 基本信息
显示患者完整信息(只读卡片)，分区:
- 个人信息区: 姓名/性别/生日/证件/语言
- 联系方式区: 电话/WhatsApp/LINE/邮箱/地址
- 医疗信息区: 血型/过敏/病史/慢性病/用药
- 监护人/紧急联系人区
- 家庭关联区: 显示关联的家庭成员患者列表(可链接跳转)
- 来源信息区: 来源/推荐人/旅游患者信息

### Tab 2: 就诊记录
时间线展示所有预约记录:
| 列标题 | 字段名 | 说明 |
|--------|--------|------|
| 就诊日期 | schedule_date | |
| 医生 | doctor.name | |
| 诊疗类型 | treatment_type | 标签 |
| 主诉 | chief_complaint | |
| 状态 | status | |
| 操作 | - | 查看详情/查看病历/查看账单 |

### Tab 3: 牙位图
嵌入牙位图组件(详见 medical-record/dental-chart.md):
- 显示患者当前牙位图全貌
- 每颗牙可点击查看状态和历史
- 快捷操作: 更新牙齿状态

### Tab 5: 影像资料
网格展示影像:
- 缩略图网格，可切换列表视图
- 筛选: 影像类型(口内照/根尖片/全景片/CBCT)
- 每张影像显示: 缩略图/标题/拍摄日期/关联牙位
- 点击查看大图+标注
- 上传新影像按钮

## 交互说明
1. 标签页切换时按需加载数据(lazy load)
2. 每个 Tab 有独立的刷新按钮
3. 支持从URL直接定位到某个Tab: /patients/:id?tab=billing
4. 患者信息编辑使用抽屉表单
5. 家庭关联: 可搜索其他患者进行关联
