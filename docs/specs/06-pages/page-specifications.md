# 页面详细设计

> 生成时间: 2026-04-15T09:37:41Z
> 执行范围: mvp
> 目标市场: 东南亚 (Southeast Asia)

## 页面清单

### P-001: 登录页
- **路由**: `/login`
- **布局**: 居中卡片
- **组件**:
  - Logo + 产品名称
  - 账号输入框
  - 密码输入框
  - 记住我 checkbox
  - 登录按钮
  - 忘记密码链接
  - 语言切换下拉
- **交互**:
  - 表单验证（必填、格式）
  - 登录中 loading 状态
  - 错误提示（toast）
  - 成功后跳转工作台

### P-002: 工作台
- **路由**: `/dashboard`
- **布局**: 侧边栏 + 顶栏 + 内容区
- **组件**:
  - 今日预约统计卡片
  - 营收统计卡片（多币种）
  - 待办事项列表
  - 预约时间线
  - 库存预警提醒
  - 商城待处理订单
  - 快捷操作按钮组
- **数据源**: Dashboard API

### P-003: 预约日历
- **路由**: `/appointments/calendar`
- **布局**: 主内容区
- **组件**:
  - 日历组件 (日/周/月切换)
  - 预约卡片 (显示患者名、医生、时段)
  - 新建预约快捷按钮
  - 筛选栏 (医生、诊室)
- **交互**:
  - 拖拽调整预约时间
  - 点击空白时段创建预约
  - 点击预约查看详情

### P-004: 预约列表
- **路由**: `/appointments/list`
- **布局**: 表格页
- **组件**:
  - 搜索栏
  - 筛选器 (状态、医生、日期范围)
  - 数据表格
  - 分页控件
  - 批量操作按钮
- **表格列**: 预约时间、患者、医生、诊疗类型、状态、操作

### P-005: 患者列表
- **路由**: `/patients/list`
- **布局**: 表格页
- **组件**:
  - 搜索栏 (姓名、手机号)
  - 标签筛选
  - 数据表格
  - 分页控件
- **表格列**: 姓名、性别、年龄、手机号、最近就诊、标签、操作

### P-006: 患者详情
- **路由**: `/patients/:id`
- **布局**: 详情页 (标签页)
- **标签页**:
  - 基本信息
  - 就诊记录
  - 费用记录
  - 病历记录
  - 影像资料

### P-007: 收费页面
- **路由**: `/billing/charge/:visit_id`
- **布局**: 两栏 (左: 费用项, 右: 汇总)
- **组件**:
  - 费用项目选择器
  - 已选项目列表 (可调数量)
  - 折扣输入
  - 币种选择
  - 费用汇总（多币种显示）
  - 支付方式选择 (GrabPay/GCash/OVO/银行转账/现金/刷卡)
  - 确认收费按钮

### P-008: 仓库列表
- **路由**: `/warehouse/list`
- **布局**: 表格页
- **组件**:
  - 仓库卡片视图 / 列表视图切换
  - 新建仓库按钮
  - 仓库状态统计
- **卡片信息**: 仓库名称、地址、库位数量、库存总量、负责人

### P-009: 入库管理
- **路由**: `/warehouse/stock-in`
- **布局**: 表格页 + 创建表单
- **组件**:
  - 入库类型切换 (采购入库/退货入库/调拨入库)
  - 入库记录表格
  - 新建入库单按钮
- **入库单表单**:
  - 选择仓库
  - 选择供应商（采购入库时）
  - 添加商品明细 (商品/批次/数量/单价)
  - 备注
  - 提交/暂存

### P-010: 出库管理
- **路由**: `/warehouse/stock-out`
- **布局**: 表格页 + 创建表单
- **组件**:
  - 出库类型切换 (销售出库/领用出库/调拨出库)
  - 出库记录表格
  - 新建出库单按钮

### P-011: 库存盘点
- **路由**: `/warehouse/inventory-check`
- **布局**: 表格页
- **组件**:
  - 盘点单列表
  - 创建盘点按钮
  - 盘点状态筛选
  - 盘点差异汇总

### P-012: 采购管理
- **路由**: `/warehouse/purchase`
- **布局**: 表格页
- **组件**:
  - 采购申请列表
  - 采购订单列表
  - 新建采购申请
  - 审批流程展示
  - 到货验收入口

### P-013: 供应商管理
- **路由**: `/warehouse/suppliers`
- **布局**: 表格页
- **组件**:
  - 供应商列表 (名称/联系方式/评级/合同状态)
  - 新建供应商
  - 供应商详情 (采购历史/评级/合同)

### P-014: 商品管理 (商城后台)
- **路由**: `/mall/products`
- **布局**: 表格页
- **组件**:
  - 商品列表 (名称/分类/价格/库存/状态)
  - 商品搜索与筛选
  - 新建商品按钮
  - 批量上下架

### P-015: 商品编辑
- **路由**: `/mall/products/:id/edit`
- **布局**: 表单页 (多步骤)
- **步骤**:
  - 基本信息 (名称/分类/品牌/描述)
  - 规格与 SKU (规格组合/价格/库存)
  - 图片上传
  - 发布设置

### P-016: 商城订单管理
- **路由**: `/mall/orders`
- **布局**: 表格页
- **组件**:
  - 订单列表 (订单号/客户/金额/状态/时间)
  - 状态筛选 (待支付/待发货/已发货/已完成/退换中)
  - 订单详情侧栏
  - 批量发货

### P-017: 商城前台 (B2C 患者端)
- **路由**: `/shop`
- **布局**: 商城布局 (顶栏+分类+商品网格)
- **页面**:
  - 首页 (推荐/分类/促销)
  - 商品列表
  - 商品详情
  - 购物车
  - 结算 (地址/支付)
  - 订单列表
  - 订单详情

### P-018: 系统设置
- **路由**: `/settings`
- **布局**: 左侧菜单 + 右侧内容
- **子页面**:
  - 机构信息设置
  - 员工管理
  - 角色权限
  - 费用项目
  - 币种设置
  - 支付渠道配置
  - 语言设置
  - 系统参数

## 通用 UI 规范

### 颜色系统
- 主色: #2563EB (蓝色)
- 成功: #16A34A
- 警告: #D97706
- 错误: #DC2626
- 中性: #6B7280

### 字体
- 标题: 16-24px, font-weight: 600
- 正文: 14px, font-weight: 400
- 辅助: 12px, font-weight: 400
- 多语言: 使用系统字体栈确保各语言显示正确

### 间距
- 页面内边距: 24px
- 卡片间距: 16px
- 组件内间距: 12px
