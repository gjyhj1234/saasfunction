## 状态机/流转

```
┌──────────┐
│  active  │ (活跃，正常合作中)
└────┬─────┘
     │ 停用                  拉黑
     ▼                      ▼
┌──────────┐          ┌─────────────┐
│ disabled │          │ blacklisted │ (黑名单)
└────┬─────┘          └──────┬──────┘
     │ 启用                  │ 移出黑名单
     ▼                      ▼
┌──────────┐          ┌──────────┐
│  active  │          │ disabled │
└──────────┘          └──────────┘
```

### 状态流转规则

| 当前状态 | 可流转至 | 触发条件 | 说明 |
|----------|----------|----------|------|
| `active` | `disabled` | 点击 [停用] | 需确认; 停用后不出现在采购单供应商选择中 |
| `active` | `blacklisted` | 点击 [拉黑] | 需确认 + 填写拉黑原因; 红色警告弹窗; 需 `supplier:blacklist` 权限 |
| `disabled` | `active` | 点击 [启用] | 无需确认; 直接启用 |
| `blacklisted` | `disabled` | 点击 [移出黑名单] | 需确认; 恢复为停用状态 (非直接活跃); 需 `supplier:blacklist` 权限 |

## 交互说明

### INT-01: 页面加载

| 步骤 | 行为 |
|------|------|
| 1 | 页面加载时请求供应商列表 (默认筛选: status=active) |
| 2 | 并行请求: 供应商列表 + 国家枚举 |
| 3 | 表格使用骨架屏 (Skeleton) 作为加载占位 |
| 4 | 检查 URL query 参数，如有筛选条件则使用 URL 中的条件 |

### INT-02: 搜索交互

| 步骤 | 行为 |
|------|------|
| 1 | 输入关键词，300ms 防抖后自动搜索 |
| 2 | 搜索时表格显示 loading 状态 |
| 3 | 搜索结果高亮匹配文字 |
| 4 | 清空搜索框 → 恢复原筛选结果 |

### INT-03: 新建供应商

| 步骤 | 行为 |
|------|------|
| 1 | 点击 [新建供应商] → 打开新建供应商抽屉 (从右侧滑入, 720px) |
| 2 | 抽屉分区显示: 基本信息 / 银行信息 / 合同信息 / 资质文件 |
| 3 | 填写基本信息; 编码输入时实时查重 |
| 4 | 选择国家后自动判断 is_domestic |
| 5 | 上传资质文件 (拖拽或点击; 支持多文件) |
| 6 | 点击 [保存] → 表单验证 → 调用创建 API |
| 7 | 成功 → Toast "供应商创建成功"; 关闭抽屉; 刷新列表 |
| 8 | 失败 → Toast 错误提示; 抽屉保持打开 |

### INT-04: 编辑供应商

| 步骤 | 行为 |
|------|------|
| 1 | 点击 [编辑] → 打开编辑供应商抽屉，预填当前数据 |
| 2 | 供应商编码字段只读不可修改 |
| 3 | 已上传的资质文件显示缩略图/文件名; 可删除或追加上传 |
| 4 | 点击 [保存] → 调用更新 API |
| 5 | 成功 → Toast "保存成功"; 关闭抽屉; 刷新列表 |

### INT-05: 查看详情抽屉

| 步骤 | 行为 |
|------|------|
| 1 | 点击供应商编码/名称 / [查看] → 打开详情抽屉 (720px) |
| 2 | 分区显示: 基本信息 / 银行信息 / 合同信息 / 采购统计 / 资质文件 |
| 3 | 采购统计显示: 总采购额、采购次数、平均交期 |
| 4 | 资质文件支持在线预览 (PDF) 和下载 |
| 5 | 底部操作栏: [编辑] [查看采购历史] [拉黑/停用/启用] |

### INT-06: 查看采购历史

| 步骤 | 行为 |
|------|------|
| 1 | 点击 [采购历史] → 跳转到 `/warehouse/purchase?supplier_id={id}` |
| 2 | 采购管理页自动筛选该供应商的所有采购单 |

### INT-07: 拉黑供应商

| 步骤 | 行为 |
|------|------|
| 1 | 点击 [拉黑] → 弹出红色警告确认弹窗 |
| 2 | 填写拉黑原因 (必填): 预设选项: 产品质量问题 / 交付不及时 / 价格欺诈 / 违反合同 / 其他 |
| 3 | 确认 → 调用拉黑 API |
| 4 | 成功 → 状态变为 blacklisted; Toast "已加入黑名单"; 行样式变灰+红色标记 |
| 5 | 该供应商不再出现在采购单的供应商选择中 |

### INT-08: 合同到期提醒

| 步骤 | 行为 |
|------|------|
| 1 | 系统每日检查供应商合同到期日 |
| 2 | 合同到期前 30 天 → 表格中合同有效期列橙色显示 + ⚠️ 图标 |
| 3 | 合同已过期 → 红色显示 + 删除线 |
| 4 | 页面加载时如有即将到期供应商 → 顶部显示提醒横幅 "{N} 个供应商合同即将到期" |

### INT-09: 导出功能

| 步骤 | 行为 |
|------|------|
| 1 | 点击 [导出] 按钮 |
| 2 | 如有选中行 → 导出选中行; 无选中行 → 导出当前筛选结果 |
| 3 | 生成 Excel 文件 (.xlsx)，包含所有表格列 + 银行信息 + 合同信息 |
| 4 | 文件名格式: `供应商管理_{YYYYMMDD_HHmmss}.xlsx` |
| 5 | 导出完成 → Toast 提示 "导出成功" |

## API 数据源

| 接口 | 方法 | 路径 | 请求参数 | 说明 |
|------|------|------|----------|------|
| 获取供应商列表 | GET | `/api/v1/suppliers` | `keyword`, `country`, `status`, `rating`, `is_domestic`, `contract_end_start`, `contract_end_end`, `sort_field`, `sort_order`, `page`, `page_size` | 分页获取供应商列表 |
| 获取供应商详情 | GET | `/api/v1/suppliers/:id` | - | 含银行信息、合同信息、采购统计、资质文件 |
| 创建供应商 | POST | `/api/v1/suppliers` | `name`, `code`, `contact_person`, `phone`, `email`, `address`, `country`, `is_domestic`, `tax_id`, `bank_name`, `bank_account`, `payment_method`, `currency`, `contract_start`, `contract_end`, `lead_time_days`, `min_order_amount`, `rating`, `qualification_docs[]` | 新建供应商 |
| 更新供应商 | PUT | `/api/v1/suppliers/:id` | 同创建参数 (code 不可修改) | 编辑供应商 |
| 停用供应商 | PATCH | `/api/v1/suppliers/:id/disable` | - | 停用供应商 |
| 启用供应商 | PATCH | `/api/v1/suppliers/:id/enable` | - | 启用供应商 |
| 拉黑供应商 | PATCH | `/api/v1/suppliers/:id/blacklist` | `reason` | 加入黑名单 |
| 移出黑名单 | PATCH | `/api/v1/suppliers/:id/unblacklist` | - | 从黑名单恢复为停用 |
| 批量停用 | POST | `/api/v1/suppliers/batch-disable` | `ids[]` | 批量停用 |
| 导出供应商 | POST | `/api/v1/suppliers/export` | 同列表筛选参数 + `ids[]` (可选) | 导出 Excel |
| 上传资质文件 | POST | `/api/v1/suppliers/:id/documents` | `file` (multipart) | 上传单个文件 |
| 删除资质文件 | DELETE | `/api/v1/suppliers/:id/documents/:doc_id` | - | 删除已上传文件 |
| 获取采购统计 | GET | `/api/v1/suppliers/:id/purchase-stats` | - | 总采购额/次数/平均交期 |
| 获取国家列表 | GET | `/api/v1/settings/countries` | - | 国家选项列表 |
