# 通用交互模式

> 版本: v2.0
> 更新时间: 2026-04-15
> 目标市场: 东南亚 (Southeast Asia)

本文档定义系统中所有页面共用的交互模式与组件规范，各页面 DSL 可通过引用本文档避免重复描述。

---

## 1. 列表页通用模式

### 1.1 页面结构

```
┌─────────────────────────────────────────────────┐
│ 面包屑导航                                        │
├─────────────────────────────────────────────────┤
│ 页面标题                         [新建] [导出]     │
├─────────────────────────────────────────────────┤
│ [搜索框]  [筛选器1 ▼] [筛选器2 ▼]  [重置]         │
├─────────────────────────────────────────────────┤
│ □ 全选  已选 N 项  [批量删除] [批量导出]           │
├─────┬───────┬──────┬──────┬──────┬──────┬───────┤
│ □   │ 列1   │ 列2  │ 列3  │ 列4  │ 列5  │ 操作  │
├─────┼───────┼──────┼──────┼──────┼──────┼───────┤
│ □   │ ...   │ ...  │ ...  │ ...  │ ...  │ [编辑]│
│     │       │      │      │      │      │ [删除]│
├─────┴───────┴──────┴──────┴──────┴──────┴───────┤
│ 共 120 条  < 1 2 3 ... 6 >   每页 [20 ▼] 条      │
└─────────────────────────────────────────────────┘
```

### 1.2 搜索栏

| 属性 | 规范 |
|------|------|
| 位置 | 表格上方左侧 |
| 组件 | Input 带搜索图标 |
| 宽度 | 320px |
| 占位符 | 根据页面内容动态设置，如 "搜索患者姓名/手机号" |
| 触发方式 | 实时搜索（输入后 300ms 防抖触发）+ 回车立即搜索 |
| 清空 | 输入框右侧显示清除按钮，点击清空并重新加载 |

### 1.3 筛选器

| 属性 | 规范 |
|------|------|
| 位置 | 搜索框右侧，同一行排列 |
| 组件 | Select / DateRangePicker / Cascader |
| 默认值 | 每个筛选器有合理默认值（如状态默认"全部"） |
| 联动 | 筛选器变化立即触发列表刷新 |
| 重置按钮 | 一键恢复所有筛选器到默认值 |

### 1.4 工具栏

| 属性 | 规范 |
|------|------|
| 位置 | 页面标题右侧 |
| 主操作 | 主按钮样式，如 [+ 新建XXX] |
| 次操作 | 次按钮样式，如 [导出]、[导入] |
| 权限控制 | 无权限的按钮不渲染 |

### 1.5 数据表格

| 属性 | 规范 |
|------|------|
| 列固定 | 左侧固定: 勾选框 + 序号列；右侧固定: 操作列 |
| 排序 | 点击表头切换升序/降序/默认，支持服务端排序 |
| 行点击 | 单击行可选中（高亮），双击进入详情（如适用） |
| 空数据 | 显示空状态插图 + "暂无数据" 提示 |
| 加载中 | 表格区域显示骨架屏（首次加载）或 Spinner（刷新） |
| 列宽调整 | 支持拖拽调整列宽 |
| 列显示/隐藏 | 提供列设置按钮，可勾选需要显示的列 |

### 1.6 行操作按钮

| 属性 | 规范 |
|------|------|
| 显示方式 | 常用操作直接显示（最多 2 个），其余收入 "更多" 下拉菜单 |
| 文字按钮 | 使用文字按钮样式，无边框 |
| 分隔符 | 操作之间用竖线分隔 |
| 权限控制 | 无权限的操作不渲染 |
| 状态控制 | 根据行数据状态动态显示/隐藏操作按钮 |

## 2. 详情页通用模式

### 2.1 页面结构

```
┌─────────────────────────────────────────────────┐
│ 面包屑: 模块 > 列表 > 详情                         │
├─────────────────────────────────────────────────┤
│ [← 返回]  标题: XXX详情         [编辑] [删除]     │
├─────────────────────────────────────────────────┤
│ ┌─────────────────────────────────────────────┐ │
│ │ 基本信息卡片                                  │ │
│ │ 字段1: 值1    字段2: 值2    字段3: 值3        │ │
│ │ 字段4: 值4    字段5: 值5    字段6: 值6        │ │
│ └─────────────────────────────────────────────┘ │
├─────────────────────────────────────────────────┤
│ [标签页1] [标签页2] [标签页3]                      │
├─────────────────────────────────────────────────┤
│ 标签页内容区域                                     │
│ （可以是子列表、时间线、表单等）                      │
└─────────────────────────────────────────────────┘
```

### 2.2 基本信息卡片

| 属性 | 规范 |
|------|------|
| 布局 | 3列栅格，响应式（移动端 1 列） |
| 字段标签 | 灰色文字 (#6B7280)，右对齐 |
| 字段值 | 黑色文字 (#111827)，左对齐 |
| 空值 | 显示 "--" |
| 状态 | 使用状态标签组件显示 |

### 2.3 标签页 (Tabs)

| 属性 | 规范 |
|------|------|
| 样式 | 线条式标签页 (line tabs) |
| 默认激活 | 第一个标签页 |
| 懒加载 | 标签页内容在切换时按需加载 |
| URL同步 | 当前标签页通过 URL query 参数同步（如 ?tab=records） |

## 3. 表单页通用模式

### 3.1 页面结构

```
┌─────────────────────────────────────────────────┐
│ 面包屑: 模块 > 列表 > 新建/编辑                    │
├─────────────────────────────────────────────────┤
│ 页面标题: 新建XXX / 编辑XXX                        │
├─────────────────────────────────────────────────┤
│ ┌─────────────────────────────────────────────┐ │
│ │ 表单区域 (白色卡片)                            │ │
│ │                                              │ │
│ │ 字段标签 *            字段标签                  │ │
│ │ [输入框]              [输入框]                 │ │
│ │                                              │ │
│ │ 字段标签 *            字段标签                  │ │
│ │ [下拉选择 ▼]          [日期选择]               │ │
│ │                                              │ │
│ └─────────────────────────────────────────────┘ │
├─────────────────────────────────────────────────┤
│                          [取消]  [保存草稿] [提交] │
└─────────────────────────────────────────────────┘
```

### 3.2 表单字段规范

| 属性 | 规范 |
|------|------|
| 布局 | 2列栅格（宽表单），1列（窄表单/抽屉内） |
| 标签位置 | 顶部对齐 (label on top) |
| 必填标识 | 红色 * 在标签文字后面 |
| 验证时机 | 失焦时验证 (onBlur) + 提交时全量验证 |
| 错误提示 | 输入框边框变红，下方显示红色错误文字 |
| 关联字段 | 联动字段变化时自动清空被联动字段值 |

### 3.3 操作按钮

| 属性 | 规范 |
|------|------|
| 位置 | 页面底部右对齐，固定在底部（表单滚动时可见） |
| 主操作 | [提交/保存] — 主按钮样式 |
| 次操作 | [取消] — 次按钮样式，返回上一页 |
| 草稿 | [保存草稿] — 次按钮样式（如适用） |
| 防重复提交 | 点击后按钮进入 loading 状态，禁止重复点击 |
| 未保存提醒 | 表单有修改但未保存时，离开页面弹出确认弹窗 |

## 4. 搜索交互

### 4.1 实时搜索

| 属性 | 规范 |
|------|------|
| 防抖延迟 | 300ms |
| 最小字符 | 输入 >= 1 个字符后开始搜索 |

| 搜索图标 | 输入框左侧显示搜索图标 |
| 加载指示 | 搜索进行中输入框右侧显示 Spinner |

### 4.2 高级搜索

| 属性 | 规范 |
|------|------|
| 触发方式 | 搜索栏旁显示 "高级搜索" 链接 |
| 展开方式 | 点击展开折叠面板，显示更多搜索条件 |
| 字段布局 | 3列栅格排列 |
| 操作 | [搜索] [重置] 按钮 |

## 5. 分页交互

| 属性 | 规范 |
|------|------|
| 位置 | 表格下方右对齐 |
| 总数显示 | "共 N 条" 显示在左侧 |
| 页码 | 显示首页、末页、当前页前后各 2 页，中间省略号 |
| 每页条数 | 下拉选择: 10 / 20（默认）/ 50 / 100 |
| 跳转 | 提供 "前往第 __ 页" 快速跳转输入框 |
| URL同步 | 当前页码和每页条数通过 URL query 参数同步 |
| 切换保持 | 切换每页条数后重置到第 1 页 |

## 6. 批量操作

| 属性 | 规范 |
|------|------|
| 勾选框 | 每行左侧显示勾选框 |
| 全选 | 表头勾选框为当前页全选/取消全选 |
| 已选提示 | 勾选后表格上方显示 "已选 N 项" |
| 批量按钮 | 选中 >= 1 项后，批量操作按钮激活 |
| 确认弹窗 | 批量删除等危险操作须弹出确认弹窗 |
| 结果反馈 | 操作完成后 Toast 提示成功/失败数量 |
| 清空选择 | 操作完成后自动清空选中状态 |

## 7. 删除确认

### 7.1 单条删除

```
┌────────────────────────────┐
│ ⚠️ 确认删除                 │
│                            │
│ 确认删除此记录？此操作不可撤销。│
│                            │
│        [取消]  [确认删除]    │
└────────────────────────────┘
```

| 属性 | 规范 |
|------|------|
| 弹窗类型 | 确认对话框 (Confirm Dialog) |
| 标题 | "确认删除" |
| 描述 | "确认删除此XXX？此操作不可撤销。" （XXX 为实体名称） |
| 取消按钮 | 次按钮样式 |
| 确认按钮 | 危险按钮样式（红色） |
| 关联校验 | 如有关联数据（如患者有未完成预约），提示无法删除并说明原因 |

### 7.2 批量删除

| 属性 | 规范 |
|------|------|
| 描述 | "确认删除选中的 N 条记录？此操作不可撤销。" |
| 部分失败 | 如部分记录无法删除，Toast 提示 "成功删除 X 条，Y 条因关联数据无法删除" |

## 10. 加载状态

| 场景 | 加载方式 | 说明 |
|------|----------|------|
| 页面首次加载 | 骨架屏 (Skeleton) | 显示内容占位骨架，保持布局稳定 |
| 列表刷新 | Spinner 遮罩 | 表格区域半透明遮罩 + 居中 Spinner |
| 按钮提交 | 按钮 Loading | 按钮内显示 Spinner + "提交中..." |
| 下拉加载选项 | 下拉 Spinner | 下拉菜单底部显示 Spinner |
| 图片加载 | 渐进加载 | 先显示低分辨率占位，再加载完整图片 |
| 详情页加载 | 骨架屏 | 各卡片区域显示骨架屏 |

## 11. 多语言 (i18n)

### 11.1 基本规范

| 属性 | 规范 |
|------|------|
| 实现方式 | 所有用户可见文案通过 i18n key 引用，禁止硬编码 |
| Key 命名 | 采用 `module.page.element` 格式，如 `patient.list.search_placeholder` |
| 默认语言 | 英语 (en) |
| 支持语言 | 英语 (en)、泰语 (th)、越南语 (vi)、印尼语 (id)、马来语 (ms) |
| 切换方式 | 顶部导航栏语言切换下拉菜单 |
| 存储 | 用户偏好语言存储在用户设置中 |
| 日期格式 | 跟随语言设置: en=MM/DD/YYYY, th=DD/MM/YYYY, vi=DD/MM/YYYY |
| 数字格式 | 千位分隔符: en/th/vi 用逗号, id/ms 用点号 |

### 11.2 语言文件结构

```
locales/
├── en/
│   ├── common.json
│   ├── patient.json
│   ├── appointment.json
│   └── ...
├── th/
├── vi/
├── id/
└── ms/
```

## 12. 多币种

| 属性 | 规范 |
|------|------|
| 币种来源 | 跟随租户（诊所）设置 |
| 支持币种 | THB（泰铢）、VND（越南盾）、IDR（印尼盾）、MYR（马来西亚林吉特）、USD（美元） |
| 显示格式 | 币种符号 + 金额，如 ฿1,500.00、₫500,000、Rp150,000 |
| 小数位 | THB/MYR/USD: 2 位小数；VND/IDR: 0 位小数 |
| 千位分隔 | 使用逗号分隔千位 |
| 输入 | 金额输入框仅允许数字和小数点，自动格式化 |
| 汇率 | 系统不做汇率转换，每个租户使用单一币种 |

## 13. 权限控制

### 13.1 按钮级权限

| 规则 | 说明 |
|------|------|
| 无权限按钮 | 不渲染（隐藏），而非禁用 |
| 权限码格式 | `module:action`，如 `patient:create`、`patient:delete` |
| 检查时机 | 组件渲染时检查当前用户权限列表 |

### 13.2 页面级权限

| 规则 | 说明 |
|------|------|
| 无权限页面 | 跳转至 403 页面 |
| 路由守卫 | 前端路由守卫检查页面权限 |
| 菜单过滤 | 侧边栏菜单只显示有权限的模块 |

### 13.3 数据级权限

| 规则 | 说明 |
|------|------|
| 租户隔离 | 不同诊所（租户）数据完全隔离 |
| 诊所过滤 | 连锁模式下，用户只能看到被授权诊所的数据 |
| 医生过滤 | 普通医生默认只看自己的患者/预约（可配置） |

## 14. 响应式设计

| 断点 | 宽度 | 布局调整 |
|------|------|----------|
| Desktop | >= 1280px | 完整布局，侧边栏展开 |
| Tablet | 768px - 1279px | 侧边栏收起为图标模式 |
| Mobile | < 768px | 侧边栏隐藏为抽屉菜单，表格横向滚动 |

### 14.1 移动端适配

- 表格: 横向可滚动，固定列保持不变
- 表单: 从 2 列变为 1 列
- 按钮: 底部固定操作栏
- 卡片: 从多列变为单列堆叠
- 弹窗: 全屏模态框取代居中弹窗

## 15. 无障碍 (Accessibility)

| 属性 | 规范 |
|------|------|
| 键盘导航 | 所有操作可通过 Tab / Enter / Esc 完成 |
| 焦点管理 | 弹窗打开时焦点锁定在弹窗内，关闭后恢复 |
| ARIA 标签 | 所有交互组件提供 aria-label |
| 颜色对比度 | 文字与背景对比度 >= 4.5:1 (WCAG AA) |
| 表单标签 | 每个输入框必须关联 label |
