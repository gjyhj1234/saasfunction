# UI 通用标准

> 版本: v2.0
> 更新时间: 2026-04-15
> 目标市场: 东南亚 (Southeast Asia)

## 颜色系统

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | #2563EB | 品牌蓝，主操作按钮、选中状态 |
| 主色-浅 | #DBEAFE | 浅蓝背景、hover状态 |
| 成功 | #16A34A | 成功提示、已完成状态 |
| 警告 | #D97706 | 警告提示、待处理状态 |
| 错误 | #DC2626 | 错误提示、删除操作、逾期状态 |
| 中性-900 | #111827 | 标题文字 |
| 中性-700 | #374151 | 正文文字 |
| 中性-500 | #6B7280 | 辅助文字 |
| 中性-300 | #D1D5DB | 边框、分割线 |
| 中性-100 | #F3F4F6 | 背景色 |
| 白色 | #FFFFFF | 卡片/模态框背景 |

## 字体规范

| 场景 | 字号 | 字重 | 行高 |
|------|------|------|------|
| 页面标题 | 24px | 600 | 32px |
| 卡片标题 | 18px | 600 | 28px |
| 表格标题 | 16px | 600 | 24px |
| 正文 | 14px | 400 | 22px |
| 辅助文字 | 12px | 400 | 18px |
| 按钮文字 | 14px | 500 | 22px |
| 标签文字 | 12px | 500 | 18px |

- 多语言字体栈: `Inter, Noto Sans Thai, Noto Sans SC, system-ui, sans-serif`
- 泰语/越南语等需要特殊字符支持的语言，确保 Noto Sans 系列字体覆盖

## 间距规范

| 场景 | 数值 |
|------|------|
| 页面外边距 | 24px |
| 卡片间距 | 16px |
| 表单字段间距 | 16px |
| 组件内间距 | 12px |
| 按钮内间距 | 8px 16px |
| 标签内间距 | 4px 8px |
| 图标与文字间距 | 8px |

## 按钮规范

| 类型 | 样式 | 使用场景 |
|------|------|----------|
| 主按钮 | 实心蓝色 (#2563EB) 白色文字，圆角 6px | 页面主操作：新建、保存、确认 |
| 次按钮 | 白色底 蓝色边框 蓝色文字 | 次要操作：取消、返回 |
| 危险按钮 | 实心红色 (#DC2626) 白色文字 | 删除、作废等不可逆操作 |
| 文字按钮 | 无边框 蓝色文字 | 链接型操作：查看详情、编辑 |
| 禁用按钮 | 灰色底 (#E5E7EB) 灰色文字 | 条件不满足时 |

## 表格规范

- 表头背景: #F9FAFB，文字: #6B7280，字重: 500
- 行高: 48px
- 斑马纹: 偶数行 #F9FAFB
- Hover: #EFF6FF
- 选中: #DBEAFE
- 固定列: 左侧固定序号列，右侧固定操作列
- 分页: 每页默认 20 条，可选 10/20/50/100

## 表单规范

- 标签位置: 顶部对齐 (label on top)
- 必填标记: 红色 * 在标签后
- 输入框高度: 36px
- 输入框圆角: 6px
- 校验提示: 红色文字显示在输入框下方
- 禁用状态: 背景 #F3F4F6

## 状态标签

| 状态 | 颜色 | 场景示例 |
|------|------|----------|
| 进行中/已确认 | 蓝色 | 预约已确认、治疗中 |
| 已完成 | 绿色 | 预约已完成、已结清 |
| 待处理 | 橙色 | 待确认、待收款 |
| 已取消/失败 | 红色 | 预约取消、支付失败 |
| 草稿 | 灰色 | 草稿状态 |
| 逾期 | 深红 | 欠费逾期、随访逾期 |

## 模态框 / 抽屉

- 模态框: 居中弹出，最大宽度 640px，遮罩层
- 抽屉: 从右侧滑出，宽度 480px/640px，适合详情查看和编辑表单
- 确认弹窗: 标题 + 描述 + 取消/确认按钮

## Toast 消息

- 位置: 右上角
- 类型: success(绿色)/warning(橙色)/error(红色)/info(蓝色)
- 自动消失: 3秒
- 可手动关闭
