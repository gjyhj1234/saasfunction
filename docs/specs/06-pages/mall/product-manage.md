## 筛选器

| 序号 | 筛选项 | 组件类型 | 选项 | 默认值 | 说明 |
|------|--------|----------|------|--------|------|
| 1 | 状态 | Select | 全部 / on_sale(上架) / off_sale(下架) / sold_out(售罄) | 全部 | 按商品上下架状态筛选 |
| 2 | 推荐 | Select | 全部 / 是 / 否 | 全部 | 筛选是否为推荐商品 |
| 3 | 售价范围 | NumberRange | 最低价~最高价 | 空 | 输入最低/最高售价进行区间筛选; 支持多币种 (THB/VND/IDR/MYR/USD) |
| 4 | 库存范围 | NumberRange | 最低库存~最高库存 | 空 | 输入最低/最高库存进行区间筛选 |

### 筛选器按钮

| 按钮 | 类型 | 说明 |
|------|------|------|
| 查询 | 主按钮 | 应用筛选条件刷新表格 |
| 重置 | 链接按钮 | 清空所有筛选条件，恢复默认值 |

### 筛选器交互

| 行为 | 说明 |
|------|------|
| 切换筛选项 | 切换后不自动查询，需点击 [查询] 按钮 |
| 筛选状态持久化 | 筛选条件同步到 URL query 参数，刷新页面保持筛选状态 |

## 状态机/流转

```
                  ┌───────────┐
       ┌─────────│  off_sale  │◄────────┐
       │  上架    │  (下架)    │  下架    │
       ▼         └─────┬─────┘         │
┌───────────┐          │          ┌───────────┐
│  on_sale  │          │ 库存=0   │  on_sale  │
│  (上架)    │──────────┼─────────►│  (上架)    │
└───────────┘          │          └───────────┘
                       ▼
                ┌───────────┐
                │ sold_out  │
                │  (售罄)    │
                └─────┬─────┘
                      │ 补货后自动恢复
                      ▼
                ┌───────────┐
                │  on_sale  │
                │  (上架)    │
                └───────────┘
```

### 状态流转规则

| 当前状态 | 可流转至 | 触发条件 | 说明 |
|----------|----------|----------|------|
| `off_sale` | `on_sale` | 点击 [上架] 按钮 | 需校验商品信息完整 (标题、图片、价格、库存) |
| `on_sale` | `off_sale` | 点击 [下架] 按钮 | 需确认; 下架后买家不可见 |
| `on_sale` | `sold_out` | 库存变为 0 时自动触发 | 系统自动流转; 前台显示 "售罄" |
| `sold_out` | `on_sale` | 库存补充后 (stock_quantity > 0) | 库存 > 0 时自动恢复上架 |
| `off_sale` | 删除 | 点击 [删除] 按钮 | 仅下架状态可删除; 物理删除 |

## 交互说明

### INT-01: 页面加载

| 步骤 | 行为 |
|------|------|
| 1 | 页面加载时请求商品列表 (默认无特定状态筛选, 显示全部) |
| 2 | 并行请求: 商品列表 + 状态枚举 + 币种配置 |
| 3 | 表格使用骨架屏 (Skeleton) 作为加载占位 |
| 4 | 检查 URL query 参数，如有筛选条件则使用 URL 中的条件 |

### INT-02: 搜索交互

| 步骤 | 行为 |
|------|------|
| 1 | 输入关键词，300ms 防抖后自动搜索 |
| 2 | 搜索时表格显示 loading 状态 |
| 3 | 搜索结果高亮匹配文字 |
| 4 | 清空搜索框 → 恢复原筛选结果 |

### INT-03: 新建商品

| 步骤 | 行为 |
|------|------|
| 1 | 点击 [新建商品] → 跳转至 `/mall/products/create` |
| 2 | 进入商品编辑页面 (详见 product-edit.md) |

### INT-04: 编辑商品

| 步骤 | 行为 |
|------|------|
| 1 | 点击 [编辑] → 跳转至 `/mall/products/:id/edit` |
| 2 | 商品编辑页自动加载商品数据并预填表单 |

### INT-05: 上下架操作

| 步骤 | 行为 |
|------|------|
| 1 | 点击 [上架] → 弹出确认弹窗 |
| 2 | 校验商品信息完整性: 标题、至少 1 张图片、价格 > 0、库存 > 0 |
| 3 | 校验不通过 → 弹窗提示具体缺失信息，引导至编辑页 |
| 4 | 确认后调用上架 API |
| 5 | 成功 → Toast "商品已上架"; 行内状态实时更新 |
| 6 | 下架操作同理，确认后调用下架 API |

### INT-06: 推荐切换

| 步骤 | 行为 |
|------|------|
| 1 | 点击表格内 ⭐ 图标 / 行操作 [设为推荐] |
| 2 | 即时调用切换推荐 API，无需确认 |
| 3 | 成功 → 图标状态即时更新; Toast "已设为推荐" / "已取消推荐" |

### INT-07: 删除商品

| 步骤 | 行为 |
|------|------|
| 1 | 点击 [删除] → 弹出确认弹窗 (危险红色样式) |
| 2 | 校验: 仅 off_sale 状态可删除 |
| 3 | 校验: 是否有未完成订单关联 (pending/paid/shipping 状态) |
| 4 | 如有未完成订单 → 弹窗提示 "该商品有 {N} 个未完成订单，无法删除" |
| 5 | 校验通过 → 调用删除 API |
| 6 | 成功 → Toast "商品已删除"; 刷新列表 |

### INT-08: 导出功能

| 步骤 | 行为 |
|------|------|
| 1 | 点击 [导出] 按钮 |
| 2 | 如有选中行 → 导出选中行; 无选中行 → 导出当前筛选结果全部数据 |
| 3 | 生成 Excel 文件 (.xlsx)，包含所有表格列 |
| 4 | 文件名格式: `商品列表_{YYYYMMDD_HHmmss}.xlsx` |
| 5 | 导出完成 → Toast 提示 "导出成功" |

## API 数据源

| 接口 | 方法 | 路径 | 请求参数 | 说明 |
|------|------|------|----------|------|
| 获取商品列表 | GET | `/api/v1/mall/products` | `keyword`, `status`, `is_featured`, `price_min`, `price_max`, `stock_min`, `stock_max`, `sort_field`, `sort_order`, `page`, `page_size` | 分页获取商城商品列表 |
| 获取商品详情 | GET | `/api/v1/mall/products/:id` | - | 商品完整信息 |
| 创建商品 | POST | `/api/v1/mall/products` | 见 product-edit.md 表单字段 | 新建商品 |
| 更新商品 | PUT | `/api/v1/mall/products/:id` | 见 product-edit.md 表单字段 | 编辑商品信息 |
| 上架商品 | PATCH | `/api/v1/mall/products/:id/on-sale` | - | 将商品设为上架 |
| 下架商品 | PATCH | `/api/v1/mall/products/:id/off-sale` | - | 将商品设为下架 |
| 切换推荐 | PATCH | `/api/v1/mall/products/:id/featured` | `is_featured`: boolean | 切换推荐状态 |
| 删除商品 | DELETE | `/api/v1/mall/products/:id` | - | 删除下架商品; 有未完成订单则拒绝 |
| 批量上架 | PATCH | `/api/v1/mall/products/batch/on-sale` | `ids[]` | 批量上架 |
| 批量下架 | PATCH | `/api/v1/mall/products/batch/off-sale` | `ids[]` | 批量下架 |
| 批量删除 | DELETE | `/api/v1/mall/products/batch` | `ids[]` | 批量删除 |
| 批量导入 | POST | `/api/v1/mall/products/import` | `file` (multipart) | 上传 Excel 批量导入 |
| 导出商品 | POST | `/api/v1/mall/products/export` | 同列表筛选参数 + `ids[]` (可选) | 导出 Excel 文件 |
