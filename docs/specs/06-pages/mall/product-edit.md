> 版本: v2.0
> 更新时间: 2026-04-15

## 页面信息

| 属性 | 值 |
|------|-----|
| 页面名称 | 商品编辑 |
| 路由 | `/mall/products/create` (新建) / `/mall/products/:id/edit` (编辑) |
| 所属模块 | 商城 (mall) |
| 布局类型 | 侧边栏 + 顶部导航 + 全页表单 (sidebar-topbar-form) |
| 权限要求 | `mall_product:create` (新建) / `mall_product:edit` (编辑) |
| 页面标题 | `新建商品 - {产品名称}` / `编辑商品: {商品标题} - {产品名称}` |
| 分页默认 | 不适用 |

## 表单字段

### 页面底部操作栏 (固定定位)

| 序号 | 按钮文本 | 类型 | 权限 | 说明 |
|------|----------|------|------|------|
| 1 | 取消 | 默认按钮 | - | 返回商品列表; 有修改时确认弹窗 "您有未保存的修改，确定离开？" |
| 2 | 保存草稿 | 次按钮 (Secondary) | `mall_product:create` / `mall_product:edit` | 保存为 off_sale 状态，不上架 |
| 3 | 预览 | 次按钮 (Secondary) | `mall_product:view` | 打开预览弹窗，模拟前台展示效果 |
| 4 | 发布上架 | 主按钮 (Primary) | `mall_product:create` / `mall_product:edit` | 保存并设为 on_sale 状态; 需通过全部表单验证 |

### 基本信息

| 字段标签 | 字段名 | 组件类型 | 必填 | 验证规则 | 说明 |
|----------|--------|----------|------|----------|------|
| 商品标题 | `title` | Input | 是 | 2-100 字符 | 商城中显示的商品名称 |
| 副标题 | `subtitle` | Input | 否 | 最多 200 字符 | 商品简短描述; 显示在标题下方 |
| 关联产品 | `product_id` | SearchSelect | 是 | 必选 | 从产品库搜索选择; 显示产品编码+名称; 搜索支持编码和名称模糊匹配 |
| 所属分类 | - | 只读文本 | - | - | 自动从关联产品读取分类信息; 不可编辑 |

### 价格库存

| 字段标签 | 字段名 | 组件类型 | 必填 | 验证规则 | 说明 |
|----------|--------|----------|------|----------|------|
| 币种 | `currency` | Select | 是 | 仅限枚举值 | THB(泰铢) / VND(越南盾) / IDR(印尼盾) / MYR(马币) / USD(美元); 默认 THB |
| 售价 | `price` | NumberInput | 是 | >0; 最多2位小数 (VND/IDR 无小数) | 商品实际售价; 币种符号自动显示在输入框前 |
| 原价 | `original_price` | NumberInput | 否 | ≥售价; 最多2位小数 | 划线价; 用于展示折扣; 为空则前台不显示原价 |
| 库存数量 | `stock_quantity` | NumberInput | 是 | ≥0; 整数 | 可同步自仓库系统或手动设置; 输入框旁显示 [从仓库同步] 按钮 |
| 最少购买量 | `min_buy_quantity` | NumberInput | 否 | ≥1; 整数; 默认 1 | 单笔订单最少购买数量 |
| 最多购买量 | `max_buy_quantity` | NumberInput | 否 | ≥min_buy_quantity; 整数; 默认 99 | 单笔订单最多购买数量; 0 表示不限制 |

### 商品图片

| 字段标签 | 字段名 | 组件类型 | 必填 | 验证规则 | 说明 |
|----------|--------|----------|------|----------|------|
| 商品图片 | `mall_images` | ImageUploader | 是 (至少 1 张) | JPG/PNG/WebP; 单张 ≤ 5MB; 最多 10 张; 建议尺寸 800×800px | 拖拽上传/点击上传; 支持拖拽排序; 第一张自动设为主图; 鼠标悬停显示 [删除] [设为主图] 按钮 |

### 商品描述

| 字段标签 | 字段名 | 组件类型 | 必填 | 验证规则 | 说明 |
|----------|--------|----------|------|----------|------|
| 商品描述 | `description` | RichTextEditor | 否 | HTML 内容最大 50000 字符 | 支持加粗/斜体/下划线/标题/图片插入/链接/有序列表/无序列表; 图片直接上传至 CDN; 支持从剪贴板粘贴图片 |

### 规格参数

| 字段标签 | 字段名 | 组件类型 | 必填 | 验证规则 | 说明 |
|----------|--------|----------|------|----------|------|
| 规格参数 | `specifications` | DynamicKeyValue | 否 | 每项: key 1-30 字符, value 1-100 字符; 最多 20 项 | 动态键值对表格; [+ 添加规格参数] 按钮新增一行; 每行含参数名、参数值和删除按钮; 支持拖拽排序 |

### 物流配置

| 字段标签 | 字段名 | 组件类型 | 必填 | 验证规则 | 说明 |
|----------|--------|----------|------|----------|------|
| 运费模板 | `shipping_template_id` | Select | 是 | 必选 | 从运费模板列表选择; 显示模板名称和运费说明 |
| 商品重量(kg) | `weight_for_shipping` | NumberInput | 否 | ≥0; 最多3位小数 | 用于物流运费计算; 单位千克 |

### 展示设置

| 字段标签 | 字段名 | 组件类型 | 必填 | 验证规则 | 说明 |
|----------|--------|----------|------|----------|------|
| 排序值 | `sort_order` | NumberInput | 否 | ≥0; 整数; 默认 0 | 数值越小越靠前; 相同排序值按创建时间倒序 |
| 推荐商品 | `is_featured` | Switch | 否 | 默认关闭 | 开启后在商城首页推荐位展示 |

## 状态机/流转

```
┌───────────────┐
│ 新建 (表单初态) │
└───────┬───────┘
        │
   ┌────┴────┐
   ▼         ▼
┌──────┐  ┌──────┐
│草稿   │  │发布   │
│off_sale│  │on_sale│
└──┬───┘  └──────┘
   │ 后续在列表页
   │ 上架操作
   ▼
┌──────┐
│on_sale│
└──────┘
```

### 表单提交流转

| 操作 | 目标状态 | 说明 |
|------|----------|------|
| 保存草稿 | `off_sale` | 保存商品信息但不上架; 仅基本信息必填项需通过验证 |
| 发布上架 | `on_sale` | 保存并上架; 全部必填项需通过验证; 至少1张图片、价格>0、库存>0 |

## 交互说明

### INT-01: 页面加载 (新建)

| 步骤 | 行为 |
|------|------|
| 1 | 路由为 `/mall/products/create` 时进入新建模式 |
| 2 | 页面标题: "新建商品" |
| 3 | 表单字段显示默认值: 币种=THB, min_buy_quantity=1, max_buy_quantity=99, sort_order=0 |
| 4 | 并行请求: 产品列表接口 (供关联产品选择) + 运费模板列表 + 币种配置 |
| 5 | 页面底部固定操作栏: [取消] [保存草稿] [预览] [发布上架] |

### INT-02: 页面加载 (编辑)

| 步骤 | 行为 |
|------|------|
| 1 | 路由为 `/mall/products/:id/edit` 时进入编辑模式 |
| 2 | 请求商品详情 API 并预填所有表单字段 |
| 3 | 页面标题: "编辑商品: {商品标题}" |
| 4 | 关联产品选择器显示已选产品信息 |
| 5 | 图片上传区显示已有图片; 保持原始排序 |
| 6 | 富文本编辑器加载已有描述内容 |
| 7 | 规格参数表加载已有键值对 |

### INT-03: 关联产品选择

| 步骤 | 行为 |
|------|------|
| 1 | 点击关联产品 SearchSelect → 弹出产品搜索下拉 |
| 2 | 输入产品名称/编码进行搜索; 300ms 防抖 |
| 3 | 下拉列表显示: 产品编码 - 产品名称 - 分类 |
| 4 | 选择产品后: 自动填充所属分类 (只读) |
| 5 | 如关联产品有默认图片，提示 "是否导入产品图片？" |

### INT-04: 图片上传与排序

| 步骤 | 行为 |
|------|------|
| 1 | 点击 [+ 添加图片] 或拖拽文件至上传区 |
| 2 | 校验: JPG/PNG/WebP 格式; 单张 ≤ 5MB |
| 3 | 上传中显示进度条; 上传完成显示缩略图 |
| 4 | 拖拽图片调整顺序; 第一张自动标记为 "主图" |
| 5 | 悬停图片显示操作: [设为主图] [删除] |
| 6 | 删除时确认弹窗: "确认删除此图片？" |
| 7 | 已上传图片数达到 10 张时，隐藏添加按钮 |

### INT-05: 库存同步

| 步骤 | 行为 |
|------|------|
| 1 | 库存数量输入框旁显示 [从仓库同步] 按钮 |
| 2 | 点击后调用仓库库存查询 API (按关联产品) |
| 3 | 返回当前仓库可用库存量并自动填入 |
| 4 | 同步成功 → Toast "库存已同步: {N} 件" |
| 5 | 未关联产品时按钮禁用; Tooltip "请先选择关联产品" |

### INT-06: 规格参数编辑

| 步骤 | 行为 |
|------|------|
| 1 | 点击 [+ 添加规格参数] → 表格末尾新增一行 (空输入框) |
| 2 | 输入参数名和参数值; 回车后光标移至下一行 |
| 3 | 点击行尾 [🗑] 删除按钮 → 移除该行; 无需确认 |
| 4 | 拖拽行头排序手柄调整参数顺序 |
| 5 | 最多添加 20 项; 达到上限后隐藏添加按钮 |

### INT-07: 富文本编辑器

| 步骤 | 行为 |
|------|------|
| 1 | 使用富文本编辑器组件 (如 TipTap / Quill) |
| 2 | 工具栏: 加粗、斜体、下划线、H1、H2、图片、链接、有序列表、无序列表 |
| 3 | 图片插入: 点击图片按钮 → 上传图片至 CDN → 插入 img 标签 |
| 4 | 支持从剪贴板直接粘贴图片 |
| 5 | 内容变更实时保存至本地 localStorage，防止意外丢失 |

### INT-08: 表单验证

| 步骤 | 行为 |
|------|------|
| 1 | 保存草稿: 仅验证必填的基本信息 (title, product_id) |
| 2 | 发布上架: 全部验证 — 基本信息 + 至少1张图片 + 价格>0 + 库存>0 + 运费模板 |
| 3 | 验证失败 → 定位到第一个错误字段; 字段下方显示红色错误提示 |
| 4 | 区块标题显示红色错误计数标记: "基本信息 (2)" |
| 5 | 售价 ≥ 原价时显示警告 (非阻塞): "售价高于或等于原价，请确认" |

### INT-09: 保存与发布

| 步骤 | 行为 |
|------|------|
| 1 | 点击 [保存草稿] → 表单验证 (宽松) → 调用保存 API (status=off_sale) |
| 2 | 点击 [发布上架] → 表单验证 (严格) → 调用保存 API (status=on_sale) |
| 3 | 新建: POST `/api/v1/mall/products`; 编辑: PUT `/api/v1/mall/products/:id` |
| 4 | 保存成功 → Toast "商品保存成功" / "商品已发布上架"; 跳转回商品列表 |
| 5 | 保存失败 → Toast 显示错误信息; 保持当前页面 |

### INT-10: 预览功能

| 步骤 | 行为 |
|------|------|
| 1 | 点击 [预览] → 打开预览弹窗 (模态框, 375px 宽度模拟手机) |
| 2 | 弹窗中渲染商品在前台的展示效果 |
| 3 | 显示: 图片轮播 + 标题 + 价格 + 描述 + 规格参数 |
| 4 | 预览使用当前表单中的实时数据 (未保存也可预览) |
| 5 | 弹窗底部: [关闭预览] 按钮 |

### INT-11: 离开确认

| 步骤 | 行为 |
|------|------|
| 1 | 表单有修改且未保存时，导航离开触发确认弹窗 |
| 2 | 弹窗文案: "您有未保存的修改，确定离开？离开后修改将丢失" |
| 3 | [取消] 留在当前页; [确定离开] 放弃修改并跳转 |
| 4 | 适用场景: 点击 [取消]、点击面包屑、点击侧边栏菜单、浏览器后退 |

## 空状态

| 场景 | 展示内容 |
|------|----------|
| 关联产品搜索无结果 | "未找到匹配的产品" + "请检查搜索词或先创建产品" |
| 运费模板为空 | "暂无运费模板" + [前往创建] 链接 |
| 规格参数为空 | "暂无规格参数" + [添加规格参数] 按钮 |
| 商品图片为空 | 虚线占位框 + 上传图标 + "点击或拖拽上传商品图片" |

## 响应式适配

| 断点 | 布局调整 |
|------|----------|
| ≥ 1280px (桌面) | 表单区域居中, 最大宽度 960px; 各区块间距 24px; 操作栏底部固定 |
| 768-1279px (平板) | 表单宽度 100%; 各区块间距 16px; 图片上传网格 4 列 |

## API 数据源

| 接口 | 方法 | 路径 | 请求参数 | 说明 |
|------|------|------|----------|------|
| 获取商品详情 | GET | `/api/v1/mall/products/:id` | - | 编辑模式加载商品数据 |
| 创建商品 | POST | `/api/v1/mall/products` | 见下方请求体 | 新建商品 |
| 更新商品 | PUT | `/api/v1/mall/products/:id` | 见下方请求体 | 编辑保存商品 |
| 搜索产品 | GET | `/api/v1/products` | `keyword`, `page`, `page_size` | 关联产品搜索选择 |
| 上传图片 | POST | `/api/v1/upload/image` | `file` (multipart), `type=mall_product` | 上传商品图片至 CDN |
| 获取运费模板 | GET | `/api/v1/mall/shipping-templates` | - | 运费模板下拉列表 |
| 同步仓库库存 | GET | `/api/v1/warehouse/stock` | `product_id` | 获取产品在仓库的可用库存 |
