## 状态机/流转

### 购物车流转

```
┌──────────┐     加入购物车     ┌──────────────┐
│ 商品浏览  │ ───────────────► │ 购物车 (本地)  │
└──────────┘                   └──────┬───────┘
                                      │ 去结算
                                      ▼
                               ┌──────────────┐
                               │ 结算页面       │
                               └──────┬───────┘
                                      │ 提交订单
                                      ▼
                               ┌──────────────┐
                               │ 支付跳转       │
                               └──────┬───────┘
                                      │ 支付完成/失败
                                 ┌────┴────┐
                                 ▼         ▼
                          ┌──────────┐ ┌──────────┐
                          │ 支付成功  │ │ 支付失败  │
                          │ 订单页面  │ │ 重新支付  │
                          └──────────┘ └──────────┘
```

### 下单流转说明

| 步骤 | 说明 |
|------|------|
| 1 | 用户浏览商品 → 点击 [加入购物车] → 商品加入购物车 |
| 2 | 打开购物车抽屉 → 调整数量 → 点击 [去结算] |
| 3 | 填写配送信息 + 选择支付方式 → 点击 [提交订单] |
| 4 | 创建订单 (status=pending) → 跳转支付网关 |
| 5 | 支付成功 → 回调更新订单状态 → 跳转订单成功页 |
| 6 | 支付失败 → 返回重新支付页 → 可选其他支付方式重试 |

## 交互说明

### INT-01: 商城首页加载

| 步骤 | 行为 |
|------|------|
| 1 | 页面加载时并行请求: 轮播横幅 + 分类列表 + 推荐商品 + 全部商品 (第一页) |
| 2 | 轮播横幅: 自动播放间隔 5 秒; 底部圆点指示器; 支持手势左右滑动 |
| 3 | 分类导航: 横向标签栏; 默认选中 "全部" |
| 4 | 推荐商品: 显示 is_featured=true 的商品, 最多 8 个 |
| 5 | 全部商品: 默认综合排序; 分页加载 |
| 6 | 页面骨架屏: 卡片占位 Skeleton 动画 |

### INT-02: 商品搜索

| 步骤 | 行为 |
|------|------|
| 1 | 点击顶部搜索框 → 输入关键词 |
| 2 | 300ms 防抖 → 请求搜索建议 API → 下拉显示最多 8 条建议 |
| 3 | 点击建议项 / 按 Enter → 展示搜索结果列表 |
| 4 | 搜索结果使用同样的卡片网格布局 |
| 5 | 无搜索结果 → 显示空状态 "未找到相关商品" |

### INT-03: 分类筛选与排序

| 步骤 | 行为 |
|------|------|
| 1 | 点击分类标签 → 即时筛选该分类商品 |
| 2 | 商品列表重置分页; 显示 loading 骨架 |
| 3 | 排序按钮: 点击切换排序方式; 价格按钮再次点击切换升降序 |
| 4 | 分类 + 排序组合筛选; URL query 同步 |

### INT-04: 商品详情查看

| 步骤 | 行为 |
|------|------|
| 1 | 点击商品卡片 → 打开商品详情页 / 弹窗 |
| 2 | 图片轮播: 支持手势滑动; 底部圆点指示; 点击放大查看 |
| 3 | 显示: 标题、副标题、价格、原价、销量、规格参数、富文本详情 |
| 4 | 底部操作栏: 数量选择器 [-] N [+] + [加入购物车] + [立即购买] |
| 5 | 数量范围: min_buy_quantity ~ min(max_buy_quantity, stock_quantity) |

### INT-05: 加入购物车

| 步骤 | 行为 |
|------|------|
| 1 | 点击 [加入购物车] 按钮 |
| 2 | 校验: 库存是否充足; 是否超过最大购买量 |
| 3 | 如购物车已有该商品 → 累加数量; 校验累计不超过 max_buy_quantity |
| 4 | 成功 → 按钮短暂变为 ✓ 勾选状态; 顶部购物车 Badge 数量+1; Toast "已加入购物车" |
| 5 | 购物车数据存储策略: 已登录用户 → 服务端 + 本地缓存; 未登录 → 仅本地 localStorage |
| 6 | 售罄商品 → 按钮禁用; 显示 "已售罄" |

### INT-06: 购物车管理

| 步骤 | 行为 |
|------|------|
| 1 | 点击顶部 🛒 图标 → 打开购物车抽屉 (从右侧滑入, 400px) |
| 2 | 显示: 商品图片、名称、单价、数量调节器、小计、删除按钮 |
| 3 | 数量调节: [-] / [+] 按钮; 最小值 min_buy_quantity; 最大值 min(max_buy_quantity, stock) |
| 4 | 删除: 点击 [🗑] → 确认弹窗 "确认从购物车移除 {商品名称}？" |
| 5 | 底部显示: 商品合计 + 运费 + 合计金额 + [去结算] 按钮 |
| 6 | 购物车为空 → 显示空状态 "购物车是空的" + [去逛逛] 按钮 |
| 7 | 实时校验库存: 如商品库存变化 → 标记提示 "库存不足, 已调整为 {N} 件" |

### INT-07: 结算下单

| 步骤 | 行为 |
|------|------|
| 1 | 点击 [去结算] → 跳转至结算页 `/mall/store/checkout` |
| 2 | 选择配送方式: 快递 / 自提 / 诊所取; 切换时地址区域动态显示/隐藏 |
| 3 | 快递配送: 填写收货地址 (自动加载上次使用的地址) |
| 4 | 自提: 选择自提门店; 显示预计可提取时间 |
| 5 | 诊所取: 显示诊所地址; 绑定下次就诊取货 |
| 6 | 选择支付方式: GrabPay / GCash / OVO / 银行转账 / 银行卡 |
| 7 | 根据用户所在国家/币种智能推荐支付方式 |
| 8 | 订单汇总: 商品合计 + 运费 + 优惠 = 应付金额 |
| 9 | 可选填订单备注 |

### INT-08: 提交订单与支付

| 步骤 | 行为 |
|------|------|
| 1 | 点击 [提交订单] → 表单验证 (配送信息 + 支付方式) |
| 2 | 调用创建订单 API → 返回订单信息 |
| 3 | 调用支付发起 API → 获取支付跳转 URL / 支付参数 |
| 4 | GrabPay/GCash/OVO: 跳转至支付平台完成支付 |
| 5 | 银行卡: 弹出安全输入框填写卡号信息 |
| 6 | 银行转账: 显示收款账户信息; 用户转账后上传转账凭证 |
| 7 | 支付成功回调 → 跳转订单成功页; 显示订单号和预计送达时间 |
| 8 | 支付失败 → 提示失败原因; 可重新选择支付方式或取消订单 |
| 9 | 提交过程中按钮显示 loading 状态; 防止重复提交 |

### INT-09: 无限滚动加载

| 步骤 | 行为 |
|------|------|
| 1 | 全部商品区域采用无限滚动 (Infinite Scroll) 加载 |
| 2 | 滚动距底部 200px 时自动请求下一页 |
| 3 | 加载中: 底部显示 loading 旋转动画 |
| 4 | 加载完毕: 显示 "已经到底了~" |
| 5 | 可选提供 [加载更多] 按钮作为手动触发方式 |

### INT-10: 轮播横幅交互

| 步骤 | 行为 |
|------|------|
| 1 | 自动播放: 每 5 秒切换一张; 用户交互后暂停自动播放 10 秒 |
| 2 | 手势滑动: 支持左右滑动切换 |
| 3 | 圆点指示器: 当前幻灯片对应圆点高亮 |
| 4 | 点击横幅: 跳转至配置的目标链接 (商品详情/分类页/活动页) |

## API 数据源

| 接口 | 方法 | 路径 | 请求参数 | 说明 |
|------|------|------|----------|------|
| 获取轮播横幅 | GET | `/api/v1/mall/store/banners` | - | 首页轮播图列表 |
| 获取分类列表 | GET | `/api/v1/mall/store/categories` | - | 商品分类导航列表 |
| 获取推荐商品 | GET | `/api/v1/mall/store/featured` | `limit` | 推荐商品列表; 默认8个 |
| 获取商品列表 | GET | `/api/v1/mall/store/products` | `keyword`, `category_id`, `sort_field`, `sort_order`, `page`, `page_size` | 分页商品列表; 仅返回 on_sale 商品 |
| 获取商品详情 | GET | `/api/v1/mall/store/products/:id` | - | 商品详细信息 (前台视角) |
| 搜索建议 | GET | `/api/v1/mall/store/search/suggest` | `keyword` | 搜索关键词建议; 最多8条 |
| 获取购物车 | GET | `/api/v1/mall/cart` | - | 当前用户的购物车内容 |
| 加入购物车 | POST | `/api/v1/mall/cart/items` | `mall_product_id`, `quantity` | 添加商品到购物车 |
| 更新购物车数量 | PUT | `/api/v1/mall/cart/items/:item_id` | `quantity` | 修改购物车商品数量 |
| 删除购物车商品 | DELETE | `/api/v1/mall/cart/items/:item_id` | - | 移除购物车中的商品 |
| 获取运费计算 | POST | `/api/v1/mall/shipping/calculate` | `items[]`, `address` | 根据商品和地址计算运费 |
| 获取自提门店 | GET | `/api/v1/mall/pickup-locations` | - | 可自提门店列表 |
| 创建订单 | POST | `/api/v1/mall/orders` | 见下方请求体 | 创建商城订单 |
| 发起支付 | POST | `/api/v1/mall/payments/initiate` | `order_id`, `payment_method`, `return_url` | 发起支付请求; 返回支付跳转信息 |
| 获取省市列表 | GET | `/api/v1/regions` | `country_code`, `parent_id` | 地址联动: 省/市/区列表 |
