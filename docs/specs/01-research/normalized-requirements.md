# 规范化需求清单

> 版本: v2.0
> 更新时间: 2026-04-15
> 执行范围: mvp
> 目标市场: 东南亚 (Southeast Asia)

## 需求规范化说明

本文档将前一阶段的东南亚市场竞品分析和用户调研结果进行规范化整理，形成结构化的需求条目。

## 功能需求清单

### FR-001: 预约管理模块
- **优先级**: P0
- **来源**: 用户调研 + 竞品分析
- **描述**: 支持多医生、多诊室的预约排班与调度，支持牙科旅游患者在线预约
- **子功能**:
  - FR-001.1: 预约创建/修改/取消
  - FR-001.2: 预约日历视图
  - FR-001.3: 预约冲突检测
  - FR-001.4: 预约提醒通知（WhatsApp/LINE/SMS）
  - FR-001.5: 在线预约（面向牙科旅游患者）

### FR-002: 患者档案模块
- **优先级**: P0
- **来源**: 用户调研 + 竞品分析
- **描述**: 患者基本信息、就诊历史、病历的统一管理，支持多语言档案
- **子功能**:
  - FR-002.1: 患者信息 CRUD
  - FR-002.2: 就诊历史时间线
  - FR-002.3: 患者标签与分组
  - FR-002.4: 患者搜索与筛选
  - FR-002.5: 多语言患者信息支持

### FR-003: 收费结算模块
- **优先级**: P0
- **来源**: 用户调研
- **描述**: 诊疗费用计算、账单生成、收款记录，支持多币种与东南亚本地支付，含欠费管理与分期付款
- **子功能**:
  - FR-003.1: 费用项目管理
  - FR-003.2: 账单生成与打印
  - FR-003.3: 收款与退款
  - FR-003.4: 多币种支持（THB/VND/IDR/MYR/SGD/PHP/USD）
  - FR-003.5: 本地支付集成（GrabPay/GCash/OVO/银行转账）
  - FR-003.6: 私人保险对接（预留）
  - FR-003.7: 欠费管理（逾期提醒/催收跟踪）
  - FR-003.8: 分期付款（分期方案配置/分期账单生成/还款跟踪）

### FR-004: 电子病历模块
- **优先级**: P0
- **来源**: 医生调研
- **描述**: 结构化口腔病历录入与管理，支持牙位图关联
- **子功能**:
  - FR-004.1: 病历模板管理
  - FR-004.2: 牙位图标注
  - FR-004.3: 治疗方案录入
  - FR-004.4: 病历打印与导出
  - FR-004.5: 双语/多语言病历支持
  - FR-004.6: 牙位图关联（病历与牙位图数据联动）

### FR-005: 临床库存管理模块
- **优先级**: P1
- **来源**: 管理者调研
- **描述**: 诊所内部耗材与器材的日常领用管理
- **子功能**:
  - FR-005.1: 科室领用记录
  - FR-005.2: 耗材消耗统计
  - FR-005.3: 低库存预警
  - FR-005.4: 与仓储系统联动

### FR-006: 仓储管理模块 (WMS)
- **优先级**: P0
- **来源**: 管理者调研 + 竞品差异化
- **描述**: 面向口腔耗材/器材/口腔护理产品的仓库管理系统，支持多仓库、多供应商、进销存全流程，为商城系统提供库存底层支撑
- **子功能**:
  - FR-006.1: 仓库管理（多仓库、库位管理）
  - FR-006.2: 入库管理（采购入库/退货入库/调拨入库）
  - FR-006.3: 出库管理（销售出库/领用出库/调拨出库）
  - FR-006.4: 库存盘点（定期/动态盘点）
  - FR-006.5: 供应商管理（供应商档案/评级/合同）
  - FR-006.6: 采购管理（采购申请/审批/订单/到货）
  - FR-006.7: 库存预警与自动补货建议
  - FR-006.8: 批次/效期管理（耗材有效期追踪）
  - FR-006.9: 进口耗材管理（多币种采购/报关信息）
  - FR-006.10: 库存报表与分析

### FR-007: 商城系统模块
- **优先级**: P1
- **来源**: 竞品差异化 + 用户调研
- **描述**: 面向 B2C（患者购买口腔护理产品）和 B2B（诊所间耗材采购）的商城系统，与仓储模块深度集成
- **子功能**:
  - FR-007.1: 商品管理（SPU/SKU、分类、品牌）
  - FR-007.2: 商品上下架与定价
  - FR-007.3: B2C 商城前台（患者端口腔护理产品商城）
  - FR-007.4: B2B 采购平台（诊所间耗材交易）
  - FR-007.5: 订单管理（创建/支付/发货/签收/退换）
  - FR-007.6: 物流跟踪集成
  - FR-007.7: 促销与优惠券
  - FR-007.8: 商城库存与仓储系统同步
  - FR-007.9: 东南亚电商平台对接预留（Shopee/Lazada）

### FR-008: 报表与分析模块
- **优先级**: P1
- **来源**: 管理者调研
- **描述**: 经营数据统计与可视化
- **子功能**:
  - FR-008.1: 营收报表
  - FR-008.2: 患者统计
  - FR-008.3: 医生工作量统计
  - FR-008.4: 库存与采购报表
  - FR-008.5: 商城销售报表
  - FR-008.6: 自定义报表

### FR-009: 多语言与本地化模块
- **优先级**: P1
- **来源**: 东南亚市场特殊需求
- **描述**: 系统界面、通知、文档的多语言支持
- **子功能**:
  - FR-009.1: 界面多语言切换（英语/泰语/越南语/印尼语/马来语等）
  - FR-009.2: 多币种配置
  - FR-009.3: 时区支持
  - FR-009.4: 本地化日期/数字格式

### FR-010: 会员体系模块
- **优先级**: P0
- **来源**: 用户调研 + 竞品分析
- **描述**: 完整的诊所会员管理体系，支持会员等级、积分、储值卡、会员套餐与会员折扣，提升患者粘性与复购率
- **子功能**:
  - FR-010.1: 会员等级管理（等级划分/升降级规则/等级权益配置）
  - FR-010.2: 积分体系（积分获取规则/积分消耗/积分兑换）
  - FR-010.3: 储值卡管理（充值/消费/余额查询/充值赠送）
  - FR-010.4: 会员套餐（套餐创建/套餐核销/套餐有效期管理）
  - FR-010.5: 会员折扣（等级折扣/专属优惠/生日特权）

### FR-011: 欠费与应收管理模块
- **优先级**: P0
- **来源**: 用户调研 + 管理者调研
- **描述**: 对患者未结清费用进行系统化管理，包括欠费跟踪、分期方案、逾期提醒、信用额度及坏账处理
- **子功能**:
  - FR-011.1: 欠费跟踪（未结账单汇总/患者欠费明细/账龄分析）
  - FR-011.2: 分期方案管理（分期计划制定/还款日历/分期手续费配置）
  - FR-011.3: 逾期提醒（自动催收通知/多渠道提醒WhatsApp/LINE/SMS）
  - FR-011.4: 信用额度管理（患者信用评估/额度设置/超额预警）
  - FR-011.5: 坏账处理（坏账认定/核销/坏账报表）

### FR-012: 治疗方案模块
- **优先级**: P0
- **来源**: 医生调研 + 竞品分析
- **描述**: 支持多步骤治疗计划的制定与全流程管理，包括费用预估、进度追踪与患者知情同意
- **子功能**:
  - FR-012.1: 多步骤治疗计划（治疗步骤定义/排序/依赖关系）
  - FR-012.2: 费用预估（治疗方案整体报价/分步费用明细/折扣计算）
  - FR-012.3: 进度追踪（治疗阶段状态更新/完成度统计/延期处理）
  - FR-012.4: 患者知情同意（同意书模板/电子签名/签署记录管理）

### FR-013: 随访管理模块
- **优先级**: P0
- **来源**: 用户调研 + 医生调研
- **描述**: 系统化管理患者随访与复诊提醒，支持术后随访及失约跟踪，提升患者依从性
- **子功能**:
  - FR-013.1: 随访计划排程（定期复查/术后随访/自定义随访周期）
  - FR-013.2: 自动提醒通知（WhatsApp/LINE/SMS多渠道自动发送）
  - FR-013.3: 术后随访管理（术后注意事项推送/恢复情况记录/异常反馈）
  - FR-013.4: 失约追踪（未到诊标记/失约原因记录/再次邀约）

### FR-014: 牙位图管理模块
- **优先级**: P0
- **来源**: 医生调研 + 竞品分析
- **描述**: 基于 FDI 牙位标记法的可视化牙位管理，支持单颗牙齿状态记录与治疗历史追溯
- **子功能**:
  - FR-014.1: FDI 牙位标记法（国际标准牙位编号/成人乳牙牙位图）
  - FR-014.2: 牙齿状态记录（健康/龋齿/缺失/修复/种植等状态标注）
  - FR-014.3: 单牙治疗历史（按牙位查看历次治疗记录/时间线回溯）

### FR-015: 临床影像管理模块
- **优先级**: P0
- **来源**: 医生调研
- **描述**: 口腔临床影像的上传与管理，支持影像与牙位图牙位强关联、影像标注及治疗前后对比
- **子功能**:
  - FR-015.1: 影像上传（X光片/CT/口内照/面相照等多类型影像上传与归档）
  - FR-015.2: 牙位图牙位强关联（影像与具体牙位绑定/按牙位查看关联影像）
  - FR-015.3: 影像标注（影像上添加标记/注释/测量）
  - FR-015.4: 治疗前后对比（同一牙位/区域的前后影像并排对比）

### FR-016: 操作日志模块
- **优先级**: P0
- **来源**: 管理者调研 + 合规要求
- **描述**: 全面记录系统内所有用户操作及数据变更历史，满足医疗合规审计要求
- **子功能**:
  - FR-016.1: 用户操作追踪（登录/登出/功能访问/关键操作记录）
  - FR-016.2: 数据变更历史（新增/修改/删除操作记录/变更前后对比）
